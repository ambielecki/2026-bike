# Edit and Delete Rides
Adds the ability to edit and delete rides.

## General overview
From the ride details page, add a button to allow editing of basic ride information. This includes ride name and description only. 
The edit button should pop a modal that allows editing those fields with a save button.

The page should also include a delete button that pops a confirmation modal. 

## Execution
Make changes to the ride details page to include the edit and delete buttons and modals. Make necessary backend changes
