# Ride List
This document describes the ride list feature in our application.

## Implementation
The Rides link is used to display a list of rides in the application.
- This is only accessible to authenticated users.
- This should show a filterable list of rides 
- Filters should include the user's locations, filter by dates range with selectors for last week, last month, and last year.
- The list should be paginated with page size options of 10, 25, 50
- The list should show the ride's date, distance in miles, and thumbnail image.
- Each item should include a link to the ride's details page (to be implemented)
